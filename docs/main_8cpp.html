Detailed Description

Main race control application logic.

This file contains the main application flow for the race control system. It initializes all hardware modules, manages the race state, handles the countdown sequence, updates the LCD display and coordinates the interaction between buttons, sensors, LEDs and car detection.

void init_project ( void  )

Initializes all dependencies and sensors. This includes:

  • Arduino Framework: Used by some of the sensors.
  • USART: Used to output status and possible errors to PC display.
  • DHT11: Temperature and humidity sensor.
  • HX1838: IR sensor used to detect passing and identification of cars.
  • LCD Dispay: Displays track information and timings.
  • Buttons: Change display output, change colour of flag.
  • LED: Flag.
