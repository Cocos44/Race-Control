Detailed Description

Button handling interface.

This file defines the button pin mapping and exposes functions used to initialize the buttons and consume button press events.

◆ BUTTONS_button_1_pressed()

bool BUTTONS_button_1_pressed ( void  )

Checks if BUTTON 1 was pressed.

BUTTON_1 is used to cycle through LCD display information.

Returns
true if BUTTON 1 was pressed, false otherwise.

Note that we change the state of button_1_pressed. Why?

Because if I hold down the button, the program will think that I am pressing it multiple times instead of holding it.

bool BUTTONS_button_2_pressed ( void  )

Checks if BUTTON_2 was pressed.

BUTTON 2 is used to start the race countdown once. After the race starts, it can be used to change the RGB LED color.

Returns
true if BUTTON 2 was pressed, false otherwise.
