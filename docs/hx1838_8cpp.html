Detailed Description

HX1838 infrared receiver implementation.

This file implements the infrared receiver logic used for car detection. The HX1838 receiver is connected to PD2 / INT0. INT0 detects pulse edges, while Timer1 measures pulse duration in order to identify the car that emitted the IR burst.

◆ HX1838_timer1_init()

static void HX1838_timer1_init ( void  )
static

Initializes Timer1 for HX1838 pulse width measurement.

Timer1 is configured in normal mode with a prescaler of 8. With F_CPU = 16 MHz, this gives a timer frequency of:

  • 16 MHz / 8 = 2 MHz

Therefore, one Timer1 tick represents:

  • 1 / 2 MHz = 0.5 us
Note
This function only configures Timer1. The counter is reset when a new HX1838 pulse starts.
